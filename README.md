### 影巢论坛签到插件-修改
1. 获取Cookie：登录影巢论坛后，按F12打开开发者工具，在网络或应用程序选项卡中复制Cookie
2. 在插件设置中填入Cookie
3. 设置签到时间（推荐早上8点，cron表达式：`0 8 * * *`）
4. 启用插件并保存
